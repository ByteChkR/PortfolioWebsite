  • ProjectCPP-tools Editor Project

    The repository ProjectCPP-tools contains a map editor for Gobble Run. In addition to be able to create maps in less than 30 seconds, all the shader parameters are editable in the editor. The editor has been changed according to the whishes from designers to better fit their way of working. The editor has a total of 8 releases that were all deployed within the timeframe of the project.

      Most outstanding features
    • Saving Maps in custom file formats/saving maps as pieces of maps(chunks) that can be shuffled and concatenated to form a big map.
    • The map is biome based, meaning that the same ids can be used to create a map that is looking totally different visually but are the exact same gameplay wise.
    • The edior allows the parallel use of the engine and the editor to load maps on the fly.
    • start from specific sections and access to a full engine debug mode that was developed to make designers able to view every piece of their map without playing the game.
    • The whole game and editor is in combination with the ADL Libary(another private tool project).

  • Advanced Debug Logging(ADL)

    This repository is a Debugging framework written in C#, that was developed to make debugging games easier by offering a performant way to store/print logs. The system can also be used to send other things than logs.

      Most outstanding features
    • Multithreaded debug logging so almost no time gets lost when debugging game loops.
    • Flag based debug system using bitflags as masks.
    • Can be used for other languages as well(Example: CPP-tools Turkey Launcher.)
    • Static Design and detailed documentation(~90% test/comment coverage)
    • Libary split up into sublibaries. The main libary contains what is nessecary to make it run. All the extras are additions to the core libary. The two Additions up until now are a Unity Wrapper to set up the whole system in a unity editor without knowing how to code and a Custom Multithreaded debug window with color codes and other helpful features.á
    • Extensive Documentation and other helpful links to work with ADL.
